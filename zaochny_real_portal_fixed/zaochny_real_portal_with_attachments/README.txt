REAL ZAOCHNY PORTAL — сайт для заочных учеников с реальной работой между устройствами

Это уже НЕ просто HTML-сайт.
Здесь есть:
- Node.js сервер
- Express API
- SQLite база данных
- загрузка файлов
- роли teacher/student
- создание заданий преподавателем
- выполнение заданий студентом с другого устройства
- проверка и оценивание работ

--------------------------------------------------
1. КАК ЗАПУСТИТЬ
--------------------------------------------------

1) Установи Node.js:
https://nodejs.org

2) Открой папку проекта в VS Code.

3) Открой терминал в этой папке.

4) Напиши:

npm install

5) Потом:

npm start

6) Открой в браузере:

http://localhost:3000

--------------------------------------------------
2. ДЕМО-АККАУНТЫ
--------------------------------------------------

Преподаватель:
login: teacher
password: 1234

Студент:
login: student
password: 1234

Другой студент:
login: student2
password: 1234

--------------------------------------------------
3. КАК ОТКРЫТЬ С ДРУГОГО УСТРОЙСТВА
--------------------------------------------------

Важно: оба устройства должны быть в одной Wi-Fi сети.

На компьютере, где запущен сервер:

Windows:
1) Открой CMD
2) Напиши:

ipconfig

3) Найди IPv4 Address, например:
192.168.1.15

На другом телефоне/ноутбуке открой:

http://192.168.1.15:3000

Только вместо 192.168.1.15 напиши свой IP.

--------------------------------------------------
4. КАК РАБОТАЕТ ЛОГИКА
--------------------------------------------------

Преподаватель:
- входит как teacher
- создает задание
- видит ответы студентов
- ставит оценку

Студент:
- входит как student или student2
- видит задание
- отправляет текстовый ответ
- может прикрепить файл
- видит оценку после проверки

--------------------------------------------------
5. ЕСЛИ НУЖНО ЧЕРЕЗ ИНТЕРНЕТ
--------------------------------------------------

Чтобы сайт работал не только в одной Wi-Fi сети, а везде, его нужно загрузить на хостинг:
- Render
- Railway
- VPS сервер
- другой Node.js хостинг

Для учебы и демонстрации достаточно запуска через npm start.

--------------------------------------------------
ЕСЛИ ПИШЕТ, ЧТО ПАРОЛЬ НЕПРАВИЛЬНЫЙ
--------------------------------------------------

1) Останови сервер: Ctrl + C
2) Удали папку data внутри проекта
3) Запусти снова:

npm start

После этого пароли снова станут:

teacher / 1234
student / 1234
student2 / 1234

В этой исправленной версии сервер также сам обновляет пароли в базе на 1234.


--------------------------------------------------
ОБНОВЛЕНИЕ: ФАЙЛ И ФОТО К ЗАДАНИЮ
--------------------------------------------------

Теперь преподаватель при создании задания может прикрепить:
- файл задания
- фото задания

Студенты увидят кнопки:
- Открыть файл задания
- Открыть фото задания

Файлы сохраняются на сервере в папке uploads.
